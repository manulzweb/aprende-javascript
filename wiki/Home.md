# 📚 Aprende JavaScript

Bienvenido/a al wiki del repositorio **[aprende-javascript](https://github.com/manulzweb/aprende-javascript)**.

Este es un curso práctico de JavaScript pensado para aprender **desde
cero hasta temas avanzados**, con ejemplos sencillos y ejecutables.
Cada módulo del repositorio tiene su carpeta con archivos `.js` que
puedes correr y modificar; este wiki es la **guía teórica** que los
acompaña.

## 🎯 ¿Para quién es este curso?

- Personas que empiezan a programar y quieren una base sólida de JS.
- Quienes ya programan en otro lenguaje y quieren aprender JavaScript.
- Quien quiera repasar conceptos avanzados (asincronía, observables).

## 🗺️ Ruta de aprendizaje

Los módulos están numerados y pensados para seguirse **en orden**. Cada
uno asume que entiendes el anterior.

| Módulo | Tema | Nivel |
|--------|------|-------|
| [00 - Variables](00-Variables) | `var`, `let`, `const`, scope | 🟢 Básico |
| [01 - Tipos de datos](01-Tipos-de-datos) | Primitivos vs referencia | 🟢 Básico |
| [02 - Operadores](02-Operadores) | Aritméticos, comparación, lógicos | 🟢 Básico |
| [03 - Control de flujo](03-Control-de-flujo) | `if`, `switch`, bucles | 🟢 Básico |
| [04 - Funciones](04-Funciones) | Declaración, expresión, arrow, closures | 🟡 Intermedio |
| [05 - Objetos](05-Objetos) | Creación, propiedades, métodos | 🟡 Intermedio |
| [06 - Arrays](06-Arrays) | Métodos, iteración, inmutabilidad | 🟡 Intermedio |
| [07 - Asincronía](07-Asincronia) | Callbacks, promesas, async/await | 🔴 Avanzado |
| [08 - Observables](08-Observables) | Programación reactiva, RxJS | 🔴 Avanzado |

## 🚀 Cómo empezar

1. Clona el repositorio:
   ```bash
   git clone https://github.com/manulzweb/aprende-javascript.git
   cd aprende-javascript
   ```
2. Asegúrate de tener [Node.js](https://nodejs.org/) instalado:
   ```bash
   node --version
   ```
3. Entra a un módulo y ejecuta cualquier archivo:
   ```bash
   cd 00-variables
   node 1-var.js
   ```

## 💡 Cómo aprovechar este curso

- **No solo leas: ejecuta.** Corre cada archivo y cambia los valores
  para ver qué pasa.
- **Rompe las cosas a propósito.** Los errores enseñan mucho.
- **Sigue el orden.** Cada módulo construye sobre el anterior.
- Usa el menú de la derecha (_Sidebar_) para navegar entre temas.
